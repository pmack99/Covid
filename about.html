About this COVID-19 Project

Data Sources
The data that I use for this site is taken from a few different APIs. For my lead pages, I use the Covid19api. That data is sourced from Johns Hopkins CSSE. The COVID Tracking Project is my seconday source. Their data comes from state/district/territory public health authorities. For the detailed Florida county information, the API is from the Florida Department of Health Open Data project
About me
My name is Preston Mack. Although I have been a professional photographer for 25 years, I have always loved computers and software development. From 2007-2010, I was an Apple Certified Developer and created 4 iOS games. In 2019 I graduated from the UCF Coding Bootcamp where I learned the tools of a web developer. When the 2020 COVID-19 Pandemic struck, I was inspired to create a site that would help me to visualize all this data.
